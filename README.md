# LifetimeTracker-Sample
- [krzysztofzablocki/LifetimeTracker](https://github.com/krzysztofzablocki/LifetimeTracker)を試すためのサンプルコード

## Requirements
```
xcode10.x
swift 4.2
cocoa pods
```

## Getting Started
```sh
$ bundle install --path=vendor/bundle --binstubs=vendor/bin
$ bundle exec pod install
```
